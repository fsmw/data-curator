                            QUICK START
================================================================================

1. Ensure requirements are installed:
   python -m pip install -r requirements.txt

2. Launch the TUI:
   python -m src.tui

3. Start exploring:
   - Press 1-7 to jump between screens
   - Press H for keyboard shortcuts
   - Press Q to quit
